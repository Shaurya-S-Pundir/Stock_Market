
# Stock Market Project

## Project Overview
This project focuses on analyzing and predicting stock market trends using Python, FastAPI, and React. 

## Folder Structure
- `data/` : Store datasets (raw and processed)
- `src/` : Python scripts and modules
- `notebooks/` : Jupyter notebooks for exploration
- `reports/` : Analysis results, charts, and reports

## Getting Started
1. Clone the repository.
2. Install required packages: `pip install -r requirements.txt`
3. Run FastAPI backend: `uvicorn src.main:app --reload`
